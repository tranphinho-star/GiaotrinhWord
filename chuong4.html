Bảng biểu (Tables)

Tạo, chỉnh sửa, định dạng bảng. Merge/Split, Formulas, Table Styles, bảng lương và bảng điểm.

➕ 4.1 Tạo bảng

Các cách tạo bảng

  1. Insert → Table → Kéo chọn số hàng × cột (tối đa 10×8)
  2. Insert → Table → Insert Table... → Nhập số cột, số hàng
  3. Insert → Table → Draw Table → Vẽ bảng bằng chuột
  4. Insert → Table → Excel Spreadsheet → Chèn bảng tính Excel
  5. Gõ nhanh: Gõ +---+---+---+ rồi Enter → Tự tạo bảng 3 cột
💡
Di chuyển trong bảng: Tab = ô tiếp theo, Shift+Tab = ô trước. Nhấn Tab ở ô cuối cùng sẽ tạo hàng mới.

🎨 4.3 Định dạng bảng

Table Styles

Click vào bảng → Tab Design → Chọn mẫu từ Table Styles gallery. Có thể tùy chỉnh thêm:

  • Header Row – Tô đậm hàng tiêu đề
  • Banded Rows – Xen kẽ màu nền giữa các hàng
  • First Column – Tô đậm cột đầu tiên
  • Borders – Tùy chỉnh đường viền
  • Shading – Tô màu nền ô

Auto Fit (Tự điều chỉnh kích thước)

  • AutoFit Contents – Co bảng theo nội dung
  • AutoFit Window – Bảng rộng bằng trang
  • Fixed Column Width – Giữ nguyên chiều rộng cột

🔢 4.4 Công thức trong bảng

Word hỗ trợ một số công thức tính toán đơn giản trong bảng:

Cách chèn công thức

Click vào ô cần chứa kết quả

Tab Layout → Formula

Nhập công thức, ví dụ: =SUM(ABOVE)

Các công thức thường dùng

Công thứcÝ nghĩa
=SUM(ABOVE)Tổng các ô phía trên
=SUM(LEFT)Tổng các ô bên trái
=AVERAGE(ABOVE)Trung bình phía trên
=MAX(ABOVE)Giá trị lớn nhất phía trên
=MIN(ABOVE)Giá trị nhỏ nhất phía trên
=COUNT(ABOVE)Đếm số ô phía trên
=B2*C2Tham chiếu ô cụ thể (cột B, hàng 2)
⚠️
Công thức trong Word không tự cập nhật khi thay đổi dữ liệu. Nhấn F9 để cập nhật hoặc click phải → Update Field.

📊 4.5 Sắp xếp dữ liệu (Sort)

Click vào bảng → Layout → Sort:

  • Chọn cột cần sắp xếp (Sort by)
  • Kiểu: Text, Number, Date
  • Thứ tự: Ascending (tăng dần) hoặc Descending (giảm dần)
  • Check "My list has: Header row" nếu hàng đầu là tiêu đề

🎯 4.6 Bài tập thực hành

📝 BT 4.1 – Bảng chấm công (Dễ)

Tạo bảng 5 cột × 8 hàng: STT | Họ tên | Ngày công | Ngày nghỉ | Ghi chú. Hàng tiêu đề: Bold, Center, nền xanh nhạt. Merge 2 ô cuối cột "Ghi chú". Áp dụng Table Style.

✅ Đáp án:

Insert → Table 5×8 → Gõ tiêu đề → Bôi đen hàng 1 → Bold, Center, Shading = Light Blue → Chọn 2 ô cuối cột Ghi chú → Merge Cells → Design → chọn Table Style.

📝 BT 4.2 – Bảng điểm học sinh (TB)

Tạo bảng điểm: STT | Họ tên | Toán | Văn | Anh | Trung bình. Nhập 5 học sinh. Dùng =AVERAGE(LEFT) để tính trung bình. Sort theo Trung bình giảm dần.

✅ Đáp án:

Tạo bảng 6 cột × 6 hàng → Nhập dữ liệu → Cột TB: Layout → Formula → =AVERAGE(C2:E2) hoặc =AVERAGE(LEFT) → Number format: 0.0 → Lặp lại cho 5 hàng → Sort by cột TB, Descending.

📝 BT 4.3 – Bảng lương (Khó)

Tạo bảng lương: STT | Họ tên | Lương cơ bản | Phụ cấp | Bảo hiểm (8%) | Thực lãnh. Nhập 5 nhân viên. Tính Bảo hiểm = Lương CB × 8%, Thực lãnh = LCB + Phụ cấp - BH. Hàng cuối: Merge cột 1-2, ghi "TỔNG CỘNG", tính SUM.

✅ Đáp án:

Tạo bảng 6 cột × 7 hàng → Nhập dữ liệu → Cột BH: =C2*0.08 → Cột Thực lãnh: =C2+D2-E2 → Hàng cuối: Merge A7:B7, gõ "TỔNG CỘNG" → Cột C7: =SUM(ABOVE). Number format: #,##0.

📝 Ghi chú của bạn 0
