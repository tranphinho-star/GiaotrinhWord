Bài tập Tổng hợp

15 bài tập từ cơ bản đến nâng cao, tổng hợp kiến thức từ tất cả các chương. Mỗi bài có đáp án chi tiết.

🟢 Cơ bản (Bài 1-5)

📝 Bài 1 – Soạn thảo đơn giản

Soạn 1 trang A4 giới thiệu bản thân: Họ tên (Bold, 16pt, Center), thông tin cá nhân, sở thích (Bullets), mục tiêu nghề nghiệp. Font TNR 13pt, Justify, Line spacing 1.5.

Kỹ năng: Ch.1 + Ch.2 + Ch.3

✅ Đáp án:

Gõ tên → Ctrl+E, Ctrl+B, 16pt → Enter → Ctrl+J, 13pt → Gõ thông tin → Bullets cho sở thích → Paragraph: Line spacing 1.5. Lưu file.

📝 Bài 2 – Bảng thời khóa biểu

Tạo bảng thời khóa biểu 7 cột (Thứ 2 → CN) × 6 hàng (tiết 1-5 + header). Merge ô cho các môn 2 tiết. Table Style đẹp. Tiêu đề Bold, Center. Ô trống tô nền xám nhạt.

Kỹ năng: Ch.4

✅ Đáp án:

Insert Table 7×6 → Header: "Tiết\Thứ", "Thứ 2"..."CN" → Merge 2 ô cho môn 2 tiết → Design → Table Style → Chọn ô trống → Shading: Light Gray → Lưu.

📝 Bài 3 – Thư cảm ơn

Soạn thư cảm ơn A4: Logo công ty (ảnh/shape), ngày tháng (căn phải), kính gửi, nội dung 2 đoạn (Justify, First Line), ký tên (căn phải). Page Border art.

Kỹ năng: Ch.2 + Ch.3 + Ch.5 + Ch.6

✅ Đáp án:

Insert Picture/Shape cho logo → Wrap: Behind Text → Ngày: Ctrl+R → Nội dung: Justify, First Line 1.27cm → Ký tên: Ctrl+R → Design → Page Borders → Art.

📝 Bài 4 – Danh sách liên hệ

Tạo bảng danh bạ: STT, Họ tên, Điện thoại, Email, Phòng ban. 10 nhân viên. Sort theo tên A-Z. Header lặp lại ở mỗi trang (Table Properties → Row → Repeat header).

Kỹ năng: Ch.4

✅ Đáp án:

Tạo bảng 5 cột × 11 hàng → Nhập dữ liệu → Layout → Sort → Sort by Họ tên, Ascending → Chọn hàng header → Table Properties → Row → ☑ Repeat as header row.

📝 Bài 5 – Tờ rơi đơn giản

Tạo tờ rơi (flyer) A4: Landscape, 3 cột. WordArt tiêu đề. 1 ảnh minh họa mỗi cột. Text Box cho thông tin liên hệ. Đường kẻ giữa các cột.

Kỹ năng: Ch.5 + Ch.6

✅ Đáp án:

Layout → Orientation: Landscape → Columns: 3 + Line between → Insert WordArt → Insert Pictures (Square wrap) mỗi cột → Insert Text Box cho contact → Căn chỉnh.

🟡 Trung bình (Bài 6-10)

📝 Bài 6 – Bảng lương nhân viên

Tạo bảng lương 10 NV: STT, Họ tên, Lương CB, Phụ cấp, BHXH (8%), Thuế TNCN (10% nếu lương>15tr, 5% nếu ≤15tr), Thực lãnh. Dùng Formula tính. Hàng tổng cộng. Merge header "Các khoản trừ" cho cột BHXH+Thuế. Định dạng số: #,##0.

Kỹ năng: Ch.4 nâng cao

✅ Đáp án:

Tạo bảng → Merge header "Các khoản trừ" → Nhập dữ liệu → BHXH: =C2*0.08 → Thuế: tính tay hoặc dùng IF → Thực lãnh: =C2+D2-E2-F2 → SUM(ABOVE) cho tổng → Format: #,##0.

📝 Bài 7 – Báo cáo với TOC

Tạo báo cáo 5+ trang: Bìa, Mục lục tự động, 3 chương (Heading 1-3), kết luận. Header: tên báo cáo. Footer: số trang (trang bìa không đánh số). Footnotes cho thuật ngữ. Ít nhất 1 bảng + 1 hình.

Kỹ năng: Ch.6 + Ch.7

✅ Đáp án:

Trang bìa → Section Break (Next Page) → Áp dụng Heading 1/2/3 → References → TOC → Insert → Header (Different First Page) → Footer: Page Number (Start at 0) → Ctrl+Alt+F cho Footnotes → Insert Table + Picture.

📝 Bài 8 – Thư mời Mail Merge

Tạo Excel: 15 khách mời (HoTen, GioiTinh, ChucVu, DiaChi). Soạn thư mời tiệc cuối năm. Mail Merge với IF rule: GioiTinh="Nam"→"Ông", else "Bà". Preview tất cả bản ghi. Finish & Merge → Print.

Kỹ năng: Ch.8

✅ Đáp án:

Excel: tạo bảng 15 hàng → Word: Mailings → Letters → Select Recipients → Insert Merge Fields → Rules → IF GioiTinh="Nam" THEN "Ông" ELSE "Bà" → Preview → Finish & Merge.

📝 Bài 9 – Sơ đồ quy trình

Tạo sơ đồ quy trình xử lý đơn hàng (7+ bước) dùng Shapes: hình chữ nhật (bước), hình thoi (quyết định), mũi tên. Group tất cả. Phiên bản 2: dùng SmartArt Process. So sánh 2 cách.

Kỹ năng: Ch.5

✅ Đáp án:

Insert → Shapes: Rectangle cho mỗi bước, Diamond cho quyết định, Arrow connectors → Add Text → Format → Align → Group. SmartArt: Insert → SmartArt → Process → Basic Chevron → Nhập text.

📝 Bài 10 – Bản tin nội bộ

Tạo newsletter 2 trang: Banner (WordArt + Shape), nội dung 2 cột có đường kẻ, ảnh minh họa (Tight wrap), Text Box quote, bảng thống kê, Watermark "NỘI BỘ". Header: logo + tên bản tin. Footer: số trang + ngày.

Kỹ năng: Ch.2-7 tổng hợp

✅ Đáp án:

Header: logo + tên bản tin → WordArt banner → Continuous Break → Columns 2 + Line between → Insert Pictures (Tight) → Text Box quote → Insert Table → Design → Watermark → Custom "NỘI BỘ" → Footer: Page Number + Auto Date.

🔴 Nâng cao (Bài 11-15)

📝 Bài 11 – Hợp đồng mua bán hoàn chỉnh

Soạn Hợp đồng mua bán hàng hóa 3+ trang: Quốc hiệu, thông tin 2 bên (bảng ẩn viền), 10 điều khoản (Multilevel), bảng danh mục hàng hóa (có formula tính thành tiền, VAT, tổng cộng). Chữ ký 2 bên. Protect: chỉ cho phép fill in forms.

Kỹ năng: Ch.3 + Ch.4 + Ch.9 + Ch.10

✅ Đáp án:

Bảng ẩn viền cho header → Multilevel List cho điều khoản → Table cho hàng hóa: Thành tiền = SL × ĐG, VAT = Thành tiền × 10%, Tổng = SUM → Bảng ẩn viền cho chữ ký → Review → Restrict Editing → Filling in forms.

📝 Bài 12 – Luận văn mẫu

Tạo luận văn mẫu 10+ trang: Bìa ngoài (Page Border), bìa trong, trang lời cam đoan, mục lục, danh mục bảng biểu, 3 chương, tài liệu tham khảo, phụ lục. Đánh số trang: i-iv (phần đầu), 1-n (nội dung). Header khác nhau giữa các chương. Styles nhất quán.

Kỹ năng: Ch.6 + Ch.7 + Ch.9 tổng hợp

✅ Đáp án:

Section Break giữa: Bìa | Phần đầu | Nội dung. Phần đầu: Page Number format i,ii. Nội dung: format 1,2,3, Start at 1. Modify Heading styles → TOC → Table of Figures. Header: Unlink, nhập tên chương riêng cho mỗi section.

📝 Bài 13 – Nhãn & Phong bì Mail Merge

Tạo Excel 30 người. Mail Merge Labels (Avery A4 21 nhãn/trang) + Mail Merge Envelopes. Nhãn gồm: Tên, Chức vụ, Công ty, Địa chỉ. Phong bì có logo người gửi.

Kỹ năng: Ch.8 nâng cao

✅ Đáp án:

Labels: Mailings → Labels → Avery → Insert Merge Fields → Update Labels → Finish. Envelopes: Mailings → Envelopes → Return address + logo → Delivery: Merge Fields → Finish & Merge.

📝 Bài 14 – Bộ văn bản nhà nước

Soạn trọn bộ 5 loại văn bản hành chính đúng thể thức NĐ30: Công văn, Quyết định, Thông báo, Biên bản họp, Tờ trình. Tạo Macro "VanBanMau" tự động thiết lập: TNR 13pt, Justify, Margins chuẩn. Tạo Style "TieuDe_VB" cho tiêu đề văn bản.

Kỹ năng: Ch.9 + Ch.10 tổng hợp

✅ Đáp án:

Tạo Macro: Record → Ctrl+A → TNR 13pt, Justify, Margins → Stop. Tạo Style: New Style → TieuDe_VB: TNR 14pt, Bold, Center, All Caps. Soạn 5 văn bản theo mẫu Ch.10, áp dụng Macro + Style.

📝 Bài 15 – Dự án tổng hợp: Hồ sơ công ty

Tạo Company Profile chuyên nghiệp 15+ trang: Bìa đẹp (WordArt, Shapes, ảnh), mục lục tự động, giới thiệu công ty (SmartArt timeline), sơ đồ tổ chức (SmartArt Hierarchy), bảng sản phẩm/dịch vụ, thành tựu (bảng + biểu đồ), thông tin liên hệ. Styles nhất quán, Header/Footer chuyên nghiệp, Watermark logo, Track changes cho bản review.

Kỹ năng: Tất cả Ch.1-10

✅ Đáp án:

Áp dụng tổng hợp tất cả kỹ năng: Section Breaks, Heading Styles → TOC, SmartArt cho timeline + org chart, Tables cho sản phẩm, Insert Chart cho thành tựu, Page Border cho bìa, Watermark cho logo, Header/Footer nhất quán, Track Changes cho review. Đây là bài tập cuối cùng kiểm tra toàn bộ kiến thức!
