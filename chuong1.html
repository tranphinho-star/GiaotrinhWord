Giới thiệu Microsoft Word

Tìm hiểu tổng quan về Microsoft Word, giao diện Ribbon, và các thao tác cơ bản nhất.

📖 1.1 Microsoft Word là gì?

Microsoft Word là phần mềm soạn thảo văn bản (word processing) phổ biến nhất thế giới, thuộc bộ Microsoft Office. Word cho phép bạn tạo, chỉnh sửa, định dạng và in ấn các tài liệu chuyên nghiệp.

🖥️ 1.2 Giao diện Microsoft Word

Khi mở Word, bạn sẽ thấy giao diện chính gồm các thành phần sau:

📌 Các thành phần chính của giao diện

  1. Title Bar – Thanh tiêu đề, hiển thị tên file đang mở
  2. Quick Access Toolbar – Thanh công cụ nhanh (Save, Undo, Redo)
  3. Ribbon – Thanh công cụ chính, chia thành các Tab:
    • Home: Font, Paragraph, Styles, Editing
    • Insert: Tables, Pictures, Shapes, Charts, Header/Footer
    • Design: Themes, Page Background
    • Layout: Page Setup, Paragraph spacing
    • References: TOC, Footnotes, Citations
    • Mailings: Mail Merge, Envelopes, Labels
    • Review: Spelling, Comments, Track Changes
    • View: Document Views, Zoom, Navigation Pane
  4. Ruler – Thước kẻ (ngang và dọc)
  5. Document Area – Vùng soạn thảo chính
  6. Status Bar – Thanh trạng thái (số trang, số từ, ngôn ngữ, zoom)
  7. Scroll Bar – Thanh cuộn dọc và ngang
💡
Mẹo: Nhấn Ctrl + F1 để ẩn/hiện Ribbon, giúp mở rộng vùng soạn thảo.

📁 1.3 Tạo, Mở và Lưu tài liệu

Tạo tài liệu mới

Mở Word → Chọn Blank document (tài liệu trống)

Hoặc nhấn Ctrl + N để tạo nhanh tài liệu mới

Chọn template (mẫu) có sẵn nếu muốn: Resume, Letter, Report...

Mở tài liệu có sẵn

Nhấn Ctrl + O hoặc vào File → Open

Duyệt đến thư mục chứa file → Chọn file → Nhấn Open

Lưu tài liệu

🎯 1.5 Bài tập thực hành

📝 Bài tập 1.1 – Tạo tài liệu đầu tiên

Yêu cầu:

  1. Mở Microsoft Word, tạo một tài liệu mới (Blank document)
  2. Gõ nội dung sau: "Đây là tài liệu đầu tiên của tôi trong Microsoft Word. Tôi đang học cách sử dụng Word từ giáo trình online."
  3. Lưu file với tên "BaiTap1_1.docx" vào Desktop
  4. Đóng file, sau đó mở lại bằng phím tắt Ctrl + O
  5. Lưu lại file dưới định dạng PDF (File → Save As → PDF)

✅ Hướng dẫn chi tiết:

Mở Word → Click Blank document

Gõ nội dung bài vào vùng soạn thảo

Nhấn Ctrl + S → Chọn Desktop → Đặt tên BaiTap1_1 → Save

Nhấn Ctrl + W để đóng file

Nhấn Ctrl + O → Duyệt đến Desktop → Chọn BaiTap1_1.docx → Open

Nhấn F12 → Chọn "Save as type" = PDF → Save

📝 Bài tập 1.2 – Khám phá giao diện Ribbon

Yêu cầu:

  1. Mở Word, lần lượt click vào từng Tab trên Ribbon (Home, Insert, Design, Layout, References, Mailings, Review, View)
  2. Ghi chú lại 3 tính năng bạn thấy hữu ích nhất ở mỗi Tab
  3. Thêm nút Print Preview vào Quick Access Toolbar
  4. Ẩn Ribbon bằng Ctrl + F1, sau đó hiện lại
