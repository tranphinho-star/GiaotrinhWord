Mail Merge & Trộn thư

Trộn thư mời, nhãn dán, phong bì, giấy chứng nhận hàng loạt từ danh sách dữ liệu.

📨 8.1 Khái niệm Mail Merge

Mail Merge (Trộn thư) cho phép tạo nhiều tài liệu cùng nội dung nhưng khác nhau ở thông tin cá nhân (tên, địa chỉ, số tiền...) từ một mẫu duy nhất + nguồn dữ liệu.

Thành phần của Mail Merge

  • Main Document – Tài liệu mẫu (thư mời, giấy chứng nhận, nhãn...)
  • Data Source – Nguồn dữ liệu (Excel, Access, danh sách Word, Outlook)
  • Merge Fields – Trường dữ liệu chèn vào mẫu (<>, <<Địa chỉ>>)
  • Merged Documents – Kết quả: nhiều tài liệu riêng lẻ

📋 8.2 Các bước thực hiện Mail Merge

Start Mail Merge: Mailings → Start Mail Merge → chọn loại (Letters, Envelopes, Labels...)

Select Recipients: Mailings → Select Recipients → Use an Existing List → chọn file Excel/CSV

Insert Merge Fields: Soạn nội dung mẫu → đặt con trỏ → Mailings → Insert Merge Field → chọn trường (VD: «HoTen», «DiaChi»)

Preview Results: Mailings → Preview Results → duyệt qua từng bản ghi bằng mũi tên

Finish & Merge: Mailings → Finish & Merge → Edit Individual Documents / Print / Send Email

📊 8.3 Tạo Data Source

Dữ liệu mẫu trong Excel

| STT | HoTen | DiaChi | Email | |-----|----------------|----------------------|--------------------| | 1 | Nguyễn Văn A | 123 Lê Lợi, Q.1 | a@email.com | | 2 | Trần Thị B | 456 Nguyễn Huệ, Q.3 | b@email.com | | 3 | Lê Văn C | 789 Hai Bà Trưng | c@email.com |

Hàng 1 là tên cột (sẽ thành Merge Fields). Mỗi hàng sau là 1 bản ghi.

⚠️
Đảm bảo file Excel không có ô merge, không có hàng trống, và tên cột không chứa ký tự đặc biệt.

🏷️ 8.4 Labels & Envelopes

Tạo nhãn dán (Labels)

Mailings → Start Mail Merge → Labels

Chọn nhà sản xuất nhãn và mã nhãn (VD: Avery, A4 65 nhãn)

Select Recipients → chọn file dữ liệu

Insert Merge Fields vào nhãn đầu tiên

Mailings → Update Labels để áp dụng cho tất cả nhãn

Finish & Merge

🔀 8.5 Rules (Quy tắc điều kiện)

IF...THEN...ELSE

Mailings → Rules → If...Then...Else:

IF GioiTinh = "Nam" THEN "Ông" ELSE "Bà"

Kết quả tự động thay đổi theo dữ liệu: "Kính gửi Ông Nguyễn Văn A" hoặc "Kính gửi Bà Trần Thị B".

🎯 8.6 Bài tập thực hành

📝 BT 8.1 – Thư mời họp (TB)

Tạo file Excel 10 người (HoTen, ChucVu, PhongBan, Email). Tạo thư mời họp với Mail Merge. Sử dụng IF rule: nếu ChucVu="Giám đốc" thì "Trân trọng kính mời" else "Kính mời".

✅ Đáp án:

Tạo Excel → Word: Mailings → Letters → Select Recipients → Insert Merge Fields → Rules → IF ChucVu = "Giám đốc" THEN "Trân trọng kính mời" ELSE "Kính mời" → Preview → Finish & Merge → Edit Individual.

📝 BT 8.2 – Giấy chứng nhận hàng loạt (Khó)

Thiết kế mẫu "Giấy chứng nhận hoàn thành khóa học" với WordArt, border, ảnh logo. Tạo data source 20 học viên. Mail Merge để tạo 20 giấy chứng nhận cá nhân.

✅ Đáp án:

Tạo mẫu A4 landscape → WordArt tiêu đề → Page Border → Logo → Nội dung với Merge Fields: «HoTen», «NgaySinh», «KhoaHoc» → Select Recipients → Preview → Finish & Merge → Print.
