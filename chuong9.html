🔍 9.1 Find & Replace nâng cao

Ngoài tìm/thay thế thông thường (Ctrl+H), Word hỗ trợ Wildcards (ký tự đại diện) rất mạnh.

Wildcards phổ biến

Ký tựÝ nghĩaVí dụ
?1 ký tự bất kỳb?t → bat, bet, bit, but
*Chuỗi ký tự bất kỳb*t → bt, bat,boat, burnt
[abc]1 trong các ký tự[bm]at → bat, mat
[a-z]Phạm vi ký tự[a-d]og → aog, bog, cog, dog
<Đầu từ<pro → program, project
>Cuối từing> → running, going
{n}Lặp n lần[0-9]{3} → 3 chữ số liên tiếp

Thủ thuật Replace hữu ích

  • Xóa dòng trống: Find: ^p^p → Replace: ^p (lặp nhiều lần)
  • Xóa khoảng trắng thừa: Find: (2 space) → Replace: (1 space)
  • Đổi định dạng: Find: text → Replace: (để trống) → Format → Bold (chỉ đổi format)

🤖 9.2 Macros

Macro tự động hóa các thao tác lặp lại, tiết kiệm thời gian đáng kể.

Record Macro (Ghi macro)

View → Macros → Record Macro

Đặt tên macro, gán phím tắt hoặc nút toolbar

Thực hiện các thao tác muốn tự động hóa

View → Macros → Stop Recording

Ví dụ Macro hữu ích

  • Macro định dạng văn bản hành chính: Tự động set Font TNR 13pt, Justify, Line spacing 1.5, Margins chuẩn VN
  • Macro chèn header công ty: Tự động thêm logo, tên công ty, địa chỉ
  • Macro dọn dẹp: Xóa dòng trống, khoảng trắng thừa, chuyển font

🎨 9.3 Styles & Themes

Styles (Kiểu)

Styles định nghĩa bộ định dạng trọn gói (font, size, color, spacing, indent...) áp dụng 1 click:

  • Normal – Đoạn văn thông thường
  • Heading 1-9 – Các cấp tiêu đề
  • Title, Subtitle – Tiêu đề chính, phụ
  • Quote – Trích dẫn

Tạo Style mới: Home → Styles → ↘ → New Style → thiết lập và đặt tên.

Modify Style: Click phải vào Style → Modify → thay đổi.

Themes (Chủ đề)

Design → Themes: thay đổi toàn bộ bảng màu, font, effects cho tài liệu chỉ với 1 click.

📝 9.4 Track Changes & Comments

Track Changes (Theo dõi thay đổi)

Ctrl+Shift+E hoặc Review → Track Changes:

  • Khi bật: mọi chỉnh sửa được đánh dấu (thêm = gạch chân, xóa = gạch ngang)
  • Accept / Reject từng thay đổi hoặc tất cả
  • Chế độ hiển thị: All Markup, Simple Markup, No Markup, Original

Comments (Nhận xét)

Review → New Comment (hoặc Ctrl+Alt+M): thêm ghi chú vào vùng văn bản được chọn.

  • Reply comment: trả lời nhận xét
  • Resolve: đánh dấu đã giải quyết
  • Delete: xóa nhận xét

🔒 9.5 Bảo vệ tài liệu

Protect Document

  • Mark as Final – Đánh dấu hoàn tất (chỉ đọc, có thể bỏ)
  • Encrypt with Password – Mật khẩu mở file
  • Restrict Editing – Giới hạn: chỉ comment, chỉ fill form, chỉ xem
  • Add Digital Signature – Chữ ký số

File → Info → Protect Document → chọn phương thức.

🎯 9.6 Bài tập thực hành

📝 BT 9.1 – Find & Replace nâng cao (TB)

Mở file văn bản bất kỳ. Dùng Find & Replace: (1) Xóa tất cả dòng trống. (2) Thay 2+ khoảng trắng thành 1. (3) Bôi đậm tất cả số điện thoại (Wildcards: [0-9]{10}).

✅ Đáp án:

(1) Ctrl+H: Find ^p^p, Replace ^p, Replace All (lặp). (2) Find {2,} (Use wildcards), Replace . (3) Find [0-9]{10}, Replace để trống → Format → Bold.

📝 BT 9.2 – Tạo Macro (Khó)

Tạo Macro "DinhDangHanhChinh" tự động: Font TNR 13pt, Justify, Line spacing 1.5 (Exactly 22pt), Margins (trên 2, dưới 2, trái 3, phải 1.5 cm). Gán phím tắt Ctrl+Shift+H.

✅ Đáp án:

View → Macros → Record → Tên: DinhDangHanhChinh → Keyboard → Ctrl+Shift+H → Assign → Thực hiện: Ctrl+A → Font TNR 13pt → Ctrl+J → Paragraph spacing Exactly 22pt → Layout → Custom Margins → Stop Recording.
